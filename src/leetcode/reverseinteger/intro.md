#### 문제를 읽고 이해하기
https://leetcode.com/problems/reverse-integer/

input</br>
x = 123
x = -123
x = 120
x = 0


output</br>
321
-321
21
0


#### 재정의와 추상화<br>
-- 문제를 읽고 본인이 이해하기 쉬운 방식으로 해석<br>
입력수를 뒤집은수 반환, 해당수가 -2^31~2^31-1의 범위에 없으면 0 반환

#### 계획 세우기<br>
수를 뒤집으면서 범위보다 크면 0을 반환해야함. 덧셈 또는 곱셈시 범위확인 후 연산진행
