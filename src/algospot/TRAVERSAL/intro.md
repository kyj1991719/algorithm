#### 재정의와 추상화<br>
-- 문제를 읽고 본인이 이해하기 쉬운 방식으로 해석<br>
전위, 중위순회를 하였을때 후위순회값

#### 계획 세우기<br>
전위 순회 : 루트 -> 왼쪽 -> 오른쪽순으로 순회, 중위 순회 : 왼쪽 -> 루트 -> 오른쪽순으로 순회<br>
후위 순회 : 왼쪽 -> 오른쪽 -> 루트순으로 순회<br>
전위의 첫번째값이 루트, 루트를 기준으로 중위에서 왼쪽과 오른쪽의 길이 계산<br>
전위순회 배열의 크기가 한개 이상인 경우 왼쪽, 오른쪽 순으로 후위순회 재귀호출 후 루트 출력<br>
