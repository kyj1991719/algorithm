#### 기본 용어
트리 : 계층적 구조를 갖는 자료들으르 표현하기 위한 자료구조, 노드들이 간선으로 서로 연결되어 있는 자료구조<br>
부모노드 : 연결된 두 노드 중 상위 노드<br>
자식노드 : 연결된 두 노드 중 하위 노드<br>
형제노드 : 부모 노드가 같은 두 노드<br>
뿌리노드(루트) : 다른 모든 노드들을 자손으로 갖는 노드<br>
잎노드(리프) : 자식이 하나도 없는 노드<br>
노드의 깊이 : 루트에서 어떤 노드에 도달하기 위해 거쳐야 하는 간선의 수<br>
트리의 높이 : 트리에서 가장 깊숙히 있는 노드의 깊이
서브 트리 : 트리에서 한 노드와 그의 자손들을 모두 모은 트리<br>
전위 순회 : 루트 -> 왼쪽 -> 오른쪽순으로 순회<br>
중위 순회 : 왼쪽 -> 루트 -> 오른쪽순으로 순회<br>
후위 순회 : 왼쪽 -> 오른쪽 -> 루트순으로 순회<br>
